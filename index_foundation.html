Getting Started

Welcome to WebKit - 808, a fully functional, HTML 5, web drum machine modeled after the classic Roland TR-808. It takes advantage of the wonderful Web Audio API" and uses jQuery Kontrol for epic dials!

Create your own sequenced beats by clicking on the blue boxes. When you're finished your funky sequence, don't forget to download it as WAV file and use it in your productions.
Have fun!
